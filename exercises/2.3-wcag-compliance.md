# 2.3 WCAG Compliance

## Goal
Meet WCAG 2.2 Level AA requirements.

---

## Errors to Find

| # | Error | Difficulty |
|---|-------|------------|
| 2.3.1 | Images with empty `alt=""` (no description) | Easy |
| 2.3.2 | Email input with `type="text"` instead of `type="email"` | Easy |
| 2.3.3 | Insufficient contrast on secondary text | Medium |
| 2.3.4 | Missing skip-link for keyboard navigation | Hard |

---

## What to Look For

```html
<!-- Empty alt -->
<img src="speaker.jpg" alt="">

<!-- Wrong input type -->
<input type="text" placeholder="Email">

<!-- Skip link should exist -->
<a href="#main" class="skip-link">Skip to content</a>
```

```css
/* Low contrast - check with WebAIM */
color: #888; /* on white background */
```

---

## Validation

1. Run axe DevTools extension
2. Run WAVE extension
3. Check contrast with WebAIM Contrast Checker
4. Navigate entire page using only keyboard
