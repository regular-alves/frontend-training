# 1.1 Core Web Vitals

## Goals
- **LCP** < 2.5s | **CLS** < 0.1 | **INP** < 200ms

---

## Errors to Find

### LCP (Loading)

| # | Error | Difficulty |
|---|-------|------------|
| 1.1.1 | Script in `<head>` without `defer` | Easy |
| 1.1.2 | Fonts without `font-display: swap` | Medium |
| 1.1.3 | Hero image without `fetchpriority="high"` | Medium |
| 1.1.4 | CSS using `@import` instead of `<link>` | Hard |

### CLS (Stability)

| # | Error | Difficulty |
|---|-------|------------|
| 1.1.5 | Images without `width` and `height` attributes | Easy |
| 1.1.6 | Image containers without `aspect-ratio` | Medium |
| 1.1.7 | Tab content without `min-height` (shifts on switch) | Hard |

---

## Validation

1. Run Lighthouse in Chrome DevTools
2. Check Performance score and individual metrics
3. Use "Performance" tab to identify bottlenecks
