# 1.2 Design Tokens

## Goal
Zero "magic values" - all styles must use tokens from `tailwind.config.js`.

---

## Errors to Find

| # | Error | Difficulty |
|---|-------|------------|
| 1.2.1 | Hardcoded colors (`bg-[#5b4dc4]`) instead of tokens | Easy |
| 1.2.2 | Arbitrary spacing (`p-[47px]`, `mt-[13px]`) | Medium |
| 1.2.3 | Arbitrary values in `:hover` that clash visually | Medium |

---

## What to Look For

```css
/* These patterns indicate magic values: */
bg-[#hexcode]
text-[#hexcode]
p-[Npx]
m-[Npx]
rounded-[Npx]
```

Replace with tokens defined in `tailwind.config.js`.

---

## Validation

1. Search code for `[#` and `[0-9]` patterns
2. Verify all values exist as tokens in config
3. Check visual consistency (same colors, consistent spacing)
