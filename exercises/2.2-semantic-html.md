# 2.2 Semantic HTML

## Goal
Replace "div-itis" with meaningful native elements.

---

## Errors to Find

| # | Error | Difficulty |
|---|-------|------------|
| 2.2.1 | `<html>` without `lang` attribute | Easy |
| 2.2.2 | `<div>` instead of `<header>`, `<nav>`, `<main>`, `<footer>` | Easy |
| 2.2.3 | Buttons as `<div onclick>` instead of `<button>` | Easy |
| 2.2.4 | Navigation links as `<div>` instead of `<a href>` | Easy |

---

## What to Look For

```html
<!-- Missing lang -->
<html>

<!-- Div-itis -->
<div class="header">
<div class="nav">
<div class="main-content">
<div class="footer">

<!-- Fake buttons -->
<div class="button" onclick="...">

<!-- Fake links -->
<div class="nav-item" onclick="...">
```

---

## Validation

1. Run W3C HTML Validator
2. Check document outline (browser extensions)
3. Test with screen reader
